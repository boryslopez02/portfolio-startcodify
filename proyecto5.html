Safecloud - Reseller

SASS, JQUERY, ScrollRevealJS, Bootstrap, PHPMailer

Sitio web interactivo con múltiples métodos de contacto (modales, formularios), se implemento Bootstrap y SASS en la maquetación, JQUERY para agregar interación con el usuario, la librería ScrollRevealJS para animar los elementos del sitio web y conseguir un resultado más dinámico, y la librería PHPMailer para implementar el envío de correos y conectar a la empresa con el usuario a tráves de un sistema de mensajería.
