Shopify - Tienda Online

HTML, SASS, JS, Bootstrap

Diseño de tienda online ecommerce de múltiples fines, tema completo con mas de 10 vistas (404, Acerca, Blog, Catálogo, Contacto, FAQs, Inicio, Login, Registro, Productos, Estilos de Producto, Servicios, Refund policy, Terminos y condiciones). Se implemento Bootstrap y SASS para agilizar el proceso de estilizar y maquetar el sitio web, Javascript para la interacción con el usuario, activar menu, filtrado de productos, carrito, y la librería ScrollRevealJS para agregar animaciones a los elementos de la web.
