Tienda Online

HTML, SASS, JS, Bootstrap

Diseño de tienda con multiples temas, el usuario puede elegir el de su preferencia y guardarlo para visualizarlo la proxima vez que visite el sitio. Se implemento Bootstrap y SASS para agilizar el proceso de estilizar y maquetar el sitio web y Javascript para la interacción con el usuario, guardar datos en LocalStorage cambiar estilos, activar menu, filtrado de productos, carrito, etc.
